Akselerator Design System menghadirkan tooling dan dokumentasi yang dibutuhkan tim untuk membangun frontend secara konsisten tanpa mengorbankan kecepatan. Saya merancang library komponen yang dapat digunakan ulang dengan token desain tematik, mendukung berbagai framework, dan dokumentasi otomatis.

### Sorotan

- Menerapkan arsitektur token-first dengan sinkronisasi gaya Figma dan ekspor kode.
- Mengotomatisasi alur rilis dengan semantic versioning dan baseline visual Storybook.
- Mengurangi waktu onboarding engineer baru melalui contoh, panduan, dan aturan lint yang terintegrasi.

### Stack & Tanggung Jawab

- Next.js App Router untuk situs dokumentasi dan playground API interaktif.
- Tailwind CSS dengan token desain yang dikompilasi dari satu sumber kebenaran.
- Aturan lint kustom dan template untuk memastikan standar aksesibilitas.

Hasilnya adalah pengalaman yang selaras di tiga lini produk dan penurunan regresi UI sebesar 40%.
